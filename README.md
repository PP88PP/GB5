## 脚本简介

**专用于境内服务器 Geekbench 5 测试脚本**

境内从 Geekbench 官网下测试程序奇慢无比，本脚本对境内使用做了优化，让下载更快，测试更便捷。

## 使用方法

```
bash <(wget -qO- https://fastly.jsdelivr.net/gh/i-abc/GB5@main/gb5-test.sh)
```
